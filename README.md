# Welcome to the GitHub page related to the scientific publication entitled: Phylogenetic signatures of defense-induced rhizosphere microbiome assembly in the mustard family

The code used to process the data in this manuscript is divided in folders based on the type of analysis.

## Within each analysis folder there will be mainly 3 directories:
code/Code: scripts

r_objects/Objects: intermediate R objects 

results/Results: plots, dataframes, pvalues

## Getting started:

**Data:** Unfiltered and filtered phyloseq objects (CSS, rarefied, non-transformed), OTU table, mapping file, taxonomy table

**01.Preprocessing:** code for preprocessing of unfiltered phyloseq objects to get filtered and normalized phyloseq objects 

**renv.lock:** Information for package installation
